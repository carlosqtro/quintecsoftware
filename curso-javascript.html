Temario del Curso

Variables y Fundamentos

Entiende la diferencia entre let, const y var. Tipos de datos primitivos y tipado dinámico.

Lógica y Control de Flujo

Toma decisiones con if/else, operadores ternarios y domina los bucles para automatizar tareas.

Funciones y Scope

Desde funciones tradicionales hasta Arrow Functions. Comprende el alcance (Scope) y el Hoisting.

Objetos y Arrays

Estructuras complejas. Métodos esenciales como .map(), .filter() y desestructuración.

El DOM y Eventos

Haz tus páginas interactivas. Selecciona elementos HTML, modifica estilos con JS y escucha clics o teclado.

Asincronía y APIs

Nivel experto. Event Loop, Promises, Async/Await y consumo de APIs con fetch.
